EL PROYECTO TIENE AUN CIERTAS COSAS EN PROCESO QUE SE IMPLANTARAN:

- INSERCCION DE UNA PELICULA POR PARTE DE ALGUN ADMINISTADOR (INDICAR SI ES ADMIN --FALLO).
- AL INSERTAR LA PELI TABIEN TENGAMOS DE INSERTAR LA PORTADA Y EL .TORRENT.
- PANEL DE ADMINISTRACION PARA ADMINISTRAR TODAS LAS PELICULA DE LA BASE DE DATOS PARA SU MODIFICACION.

-PANEL DE VALORACION DE LA PELICULA, EL USUARIO PODRA VOTAR (UNA UNICA VEZ) LA VALORACION DE LA PELICULA
    PERO NO INSERTANDO LA PELICULA SINO CON UN ELEMENTO TIPO ESTRELLAS PARA QUE EL USUARIO INTERACTIVAMENTE PUEDA INSERTAR LA VALORACION 
    DESPUES LA BASE DE DATOS SACARIA LA MEDIA DE LAS VALORACIONES Y ASÍ EN EL DEFAULT O EN CUALQUIER BUSQUEDA PODER MOSTRAR LA VALORACION.

- QUE EL USUARIO ADEMAS DE PODER DESCARGAR EL MAGNET LINK, PUEDA DESCARGAR EL .TORRENT, SABER EL CODIGO PARA QUE EL USUARIO PUEDA DESCARGAR EL ARCHIVO Y NO
    SOLO EL MAGNET DIRECTAMENTE.

- MEJORAR LA USABILIDAD DE LA WEB: MEJORAR LA EXPERIENCIA DEL USUARIO PARA PODER NAVEGAR POR LA WEB.

- CREACION DEL METODO "PELICULAS DESTACADAS", PARA PODER MOSTRAR LAS PELICULAS QUE MAS SE DESCARGAN EN ESE MOMENTO:
    CADA VEZ QUE SE EJECUTE UNA DESCARGA PODER HACER UNA TABLA EN LA BD PARA CALCULAR CADA DESCARGA, Y DESPUES MEDIANTE EL CALCULO 
    MOSTRAR LAS 5 PELICULAS QUE MAS DESCARGAS TENGAN EN ESE MOMENTO (TAMBIEN HABRIA UN WHERE VIENDO QUE LA SUBIDA DEL ARCHIVO SEA MENOS A 2 SEMANAS).
